A sales page web application built with Groovy, and the oldest repo in my GitHub account — it dates back to November 2012. The README consists entirely of a large ASCII-art banner, which was very much the style of the time.

There isn't much to document beyond that: this comes from my Groovy-on-the-JVM era (see also my AdvancedGroovy fork from a couple of years later), when Grails-style rapid web development was my toolkit of choice. It stays public as a timestamp of where things started.
